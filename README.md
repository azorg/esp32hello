Первый проект "Hello world" (sketch) для ESP32 в Arduino IDE
============================================================

# Ссылки на документацию
 * https://docs.espressif.com/projects/arduino-esp32/en/latest/

# Скетчи:
 * `blink_led_simple` - моргание светодиодом с задержками (плохая практика, простой пример)
 * `blink_led` - моргание светодиодом с использованием прерываний от таймера

# Полезные команды:
 * `esptool.py -p /dev/ttyUSB0 chip_id` - узнать версию чипа ESP32
